QuickSteps to Run POD on Windows

It is as easy as 3 simple steps to run POD on Windows. To get POD up and running simple follow the steps below. If you are running POD on another platform other then MS Windows please follow the instructions on the download page.

 

 Install Java

First install Java 1.4 from the Sun website. Installing Java on the system will provide the machine with the latest and greatest technology and provide Java support in Internet Explorer. You can skip this step if you have already installed at least Java 1.4 on your machine.

Note: POD is not supported and will not run on early versions of the JDK. JDK 1.4.1 is available for Windows, Linux, Solaris, and also Mac OS X.
 

 Install Windows POD Setup application

Simply download the latest POD setup for windows and launch the setup application. This will install POD on your machine and place an Icon on your Desktop for easy access to the POD environment.

 

 Run POD

Launch the POD environment by double clicking on the POD icon. The POD environment will be launched on top of your existing desktop environment. Additional components may need to be downloaded from the Internet at this time. Components will be cached and subsequent launches will be performed more quickly.

Now you are ready to use the POD environment.
